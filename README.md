#  AI Career Hub

AI Career Hub is a responsive front-end website designed to help students explore Artificial Intelligence career paths. The platform provides AI roadmaps, popular AI tools, internship opportunities, project ideas, and learning resources in one place.

## Features

- Responsive Design
- Sticky Navigation Bar
- Mobile Hamburger Menu
- Dark Mode Toggle
- AI Career Roadmaps
- AI Tools Section
- Internship Listings
- AI Project Ideas
- Testimonials
- Contact Form
- Smooth Scrolling
- Hover Animations

## Technologies Used

- HTML5
- CSS3
- JavaScript
